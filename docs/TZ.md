ProkatApp
Мобильное приложение для управления пунктом проката
Техническое задание и описание продукта
Версия 1.0 | Апрель 2026

1. Общее описание продукта
ProkatApp — мобильное приложение для автоматизации учёта сдачи имущества в прокат. Предназначено для владельцев пунктов проката спортивного оборудования, инструментов и другого инвентаря.
Приложение решает задачи управления оборудованием (Склад → Свободные → Занятые), оформления заказов, контроля времени аренды, фиксации оплаты и формирования отчётности.

2. Пользователи системы
2.1 Администратор (владелец)
•	Полный доступ ко всем функциям приложения
•	Создание, редактирование и удаление карточек оборудования
•	Просмотр отчётов и аналитики
•	Управление сотрудниками и их правами
•	Настройка тарифов и категорий
•	Просмотр лога действий сотрудников
•	Управление несколькими точками проката (в платной версии)
2.2 Сотрудник проката
•	Перемещение оборудования между статусами
•	Оформление новых заказов на прокат
•	Завершение и продление заказов
•	Фиксация оплаты и ввод данных арендатора
•	Просмотр текущего списка занятого и свободного оборудования

3. Главный экран и навигация
При запуске приложения пользователь попадает на главный экран с четырьмя разделами:
Раздел	Назначение
Занятые	Оборудование в аренде. Отображается время окончания, таймер обратного отсчёта, кнопки завершения и продления.
Свободные	Оборудование, готовое к выдаче. Здесь оформляются новые заказы.
Склад	Полный каталог имущества. Управление карточками, перемещение в раздел Свободные.
Отчёты	Аналитика по заказам, доходу, нагрузке. Экспорт данных.


4. Раздел «Склад»
Центральный каталог всего имущества компании. Здесь хранятся карточки оборудования вне зависимости от статуса.
4.1 Карточка оборудования
При создании или редактировании карточки указываются:
•	Категория (спортивное, инструменты, транспорт и др.)
•	Наименование оборудования
•	Характеристики (описание, технические параметры)
•	Фотография (одно или несколько фото)
•	Тариф проката (за час, за день и другие периоды)
•	Сумма залога
•	Инвентарный номер / QR-код для быстрого поиска
•	Статус (склад / свободное / занятое)
4.2 Действия со складом
•	Добавление нового оборудования
•	Редактирование карточки (только администратор)
•	Удаление / архивирование (только администратор)
•	Перемещение оборудования в раздел Свободные (начало смены)
•	Сканирование QR-кода для мгновенного открытия карточки

5. Раздел «Свободные»
Список оборудования, доступного для выдачи в прокат в текущий момент.
5.1 Оформление нового заказа
При выборе единицы оборудования открывается форма заказа со следующими полями:
•	ФИО арендатора
•	Номер телефона
•	Фото документа арендатора (паспорт) — для фиксации ответственности
•	Время начала и планируемое время окончания аренды
•	Расчётная стоимость (вычисляется автоматически по тарифу)
•	Сумма залога и способ приёма (нал / безнал)
•	Форма оплаты аренды (нал / безнал)
•	Комментарий (необязательно)
5.2 После оформления заказа
Оборудование автоматически переходит в раздел «Занятые». Оператор получает подтверждение заказа на экране.

6. Раздел «Занятые»
Список оборудования, находящегося в аренде у клиентов в данный момент.
6.1 Отображаемая информация
•	Наименование и фото оборудования
•	ФИО арендатора и телефон
•	Время начала и окончания аренды
•	Таймер обратного отсчёта (сколько осталось до конца аренды)
•	Сумма заказа и статус оплаты
6.2 Действия с заказом
•	Завершить заказ — оборудование переходит в «Свободные», фиксируется факт возврата залога
•	Продлить заказ — ввод нового времени окончания, автоматический пересчёт стоимости
•	Начислить штраф за просрочку — автоматически при превышении времени аренды
6.3 Уведомления
•	Push-уведомление сотруднику за 15 минут до окончания аренды
•	Уведомление при истечении времени аренды (просрочка)

7. Раздел «Отчёты»
7.1 Доступные отчёты
•	Количество заказов за период (день / неделя / месяц / произвольный диапазон)
•	Сумма дохода за период
•	Топ популярного оборудования (по количеству заказов)
•	Средняя длительность аренды
•	Загруженность по часам и дням недели
•	История по конкретной единице оборудования
•	Лог действий сотрудников (кто, что и когда сделал)
7.2 Экспорт
•	Выгрузка отчёта в PDF
•	Выгрузка в Excel (CSV)
•	Отправка на email

8. Жизненный цикл оборудования
Склад  →  Свободные  →  Занятые  →  Свободные  →  Склад
Перемещение между статусами возможно только в указанном порядке. Администратор может перемещать оборудование в любом направлении. Сотрудник — только по основному маршруту.

9. QR-коды
Каждая единица оборудования получает уникальный QR-код, который можно распечатать и приклеить на инвентарь.
•	Сканирование QR открывает карточку оборудования мгновенно
•	Быстрое оформление заказа без поиска по списку
•	Быстрое завершение аренды при возврате
